Install
-------

.. code-block:: bash

    $ pip install Kuai --user


Develop
-------

.. code-block:: bash

    $ clone https://github.com/Duroktar/Kuai.git
    $ cd Kuai
    $ pip install -e .


Kuai is Fast
------------

Save in a hello.py:

.. code-block:: python

    from kuai import Kuai

    def hello(world):
        print(f"Hello, {world}!")

    Kuai.on('hello-world', hello)
    Kuai.emit('hello-world', 'Kuài')

It’s as easy as

.. code-block:: bash

    $ pip install Kuai
    $ python hello.py
     Hello, Kuài!
